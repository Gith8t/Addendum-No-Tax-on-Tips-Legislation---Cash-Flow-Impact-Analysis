# Addendum: No Tax on Tips Legislation - Cash Flow Impact Analysis

## Understanding the Proposed Changes

### Current Situation (2024)
- Servers pay income tax on all tips
- Employers pay FICA (7.65%) on reported tips
- Employers withhold employee FICA (7.65%) from paychecks
- Tips count toward minimum wage requirements

### Proposed Changes
- **Federal Level**: No federal income tax on tips
- **FICA Status**: Still unclear if Social Security/Medicare applies
- **State Level**: Varies by state adoption
- **Timeline**: Could be retroactive to 2024 or start 2025
